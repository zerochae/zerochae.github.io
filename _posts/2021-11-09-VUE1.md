### Tutorial 1 (설치,프로젝트 생성)

### Vue Cli 설치
```
$ npm install -g @vue/cli
```

입력 후에 Vue3 버전을 방향키로 선택한다.

### Vue Project 생성

```
$ vue create [Project Name]
```
프로젝트 이름은 자유롭게 작명 가능

### Vue 서버 열기

```
$ npm run serve
```
